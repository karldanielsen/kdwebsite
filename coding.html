Digital Assets Portfolio Analyser [WIP]
Estimated Completion Date: 4/9/2021 Link will be available then.

A combination of an AWS live database and stored binance API coin information will allow users to adjust, save, and analyse a cryptocurrency portfolio against the historical market data. This will be achieved through a number of numerical readouts and charts.

This is NOT a trading dashboard, it is intended for use in visually investigating the natural relationships between coins, and for discovering how to well-diversify in a market dominated by one asset, when it is often difficult to do so.

The AWS templates and code are proprietary, but will be put up if ever publicly released.

Message Me!
A simple combination of a couple of AWS resources allows you to use the text entry form below to message my phone!

If you want a response, be sure to include an email address or phone number, along with your name. Your message is encrypted throughout the whole process and not stored anywhere, so information is safe.

Genomics Algorithms
When covid-19 started, there was a lot of buzz about using genetics to treat and develop vaccines for it. This buzz had a good reason for happening, seeing as the vaccines developed were RNA vaccines, based heavily on the study and reorganization of genetic material.

This project was built to benefit a group of UCLA geneticists with little computer science background, to help them isolate and study the differences between benign coronaviruses and covid-19. The end goal was helping them target their research on the specific genes that make covid unique.

I have no idea if it directly contributed to the effort to produce a cure, but it was very satisfying knowing I was doing my part in joining the worldwide effort to end the pandemic.

File Transfer
A basic module to transfer files between two systems using ports and IP addresses via built-in C++ TCP libraries. Message validation is performed via CRC (which can be cheated given that the CRC is hard- coded and the code is public). Built primarily as an informative thing, due to the lack of real-world-applicable networking taught in college.

Also includes a separate version of the server which processes clients requests with threads when they are large enough to block server processing.
