Message Me!
A simple combination of a couple of AWS resources allows you to use the text entry form below to message my phone!

If you want a response, be sure to include an email address or phone number, along with your name. Your message is encrypted throughout the whole process and not stored anywhere, so information is safe.

If you want to set one up yourself, ask me for the AWS cloudformation template, which can build a messaging API for you with just two or three pieces of info.
Note: API and notification throttling are in effect. Any attempts to spam this messager will result in a quick shutdown at no loss to myself. Don't bother.
Genomics Algorithms
When covid-19 started, there was a lot of buzz about using genetics to treat and develop vaccines for it. This buzz had a good reason for happening, seeing as the vaccines developed were RNA vaccines, based heavily on the study and reorganization of genetic material.

This project was built to benefit a group of UCLA geneticists with little computer science background, to help them isolate and study the differences in the viral genetic code from that of other coronaviruses, so they could target their research.

I have no idea if it directly contributed to the effort to produce a cure, but it was very satisfying knowing I doing my part in joining the worldwide effort to end the pandemic.
